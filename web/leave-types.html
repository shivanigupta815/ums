Leave Types

Define leave types. Use Staff Type to separate Teaching / Non-Teaching leaves. Use Gender and Marital Status for community leaves like Maternity, Child Care, Paternity.

➕ Add New Leave Type
👥 Community Eligibility Criteria — For Group Leaves (Maternity, Child Care, Paternity etc.)

Set Gender and/or Marital Status to restrict this leave to a specific group. The leave will automatically appear on the Apply Leave page for all staff who match these criteria. Leave both as "Any" for leaves that apply to everyone.

Quick Examples:
🤱 Maternity → Female + Married
👶 Child Care → Female + Married
👨‍👧 Paternity → Male + Married
📚 Study Leave → Any + Any
